创建hexo博客过程

##步骤一
下载nodejs 和 npm
将npm改为cnpm

##步骤二

下载hexo
hexo s为开始运行博客，获得一个端口号
hexo n为new一篇新文章
利用vim修改md的内容

hexo clean 重新清理
hexo g 重新部署
hexo s 重新开始
